Room-wise Unique Object Detection

Upload Images for Processing

How it Works:

Upload one or more images, optionally providing a Room ID. The system will:

  1. Detect objects (e.g., Sofa, TV, Chair) using YOLOv8.
  2. Associate detected objects with the provided Room ID.
  3. Count unique objects within that specific room (duplicates in the same room count as one).
  4. Generate a report with unique object counts per room.
  5. Provide visualizations of detected objects with bounding boxes.

You can upload images for different rooms. The system will aggregate results and provide a comprehensive room-wise report.
